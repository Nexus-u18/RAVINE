# RAVINE - Descent Into the Deep

An immersive frontend odyssey exploring the atmospheric layers of the ocean through interactive transitions and scroll-driven animations.

# Live Demo
View the project live on Vercel: [https://ravine-kappa.vercel.app](https://ravine-kappa.vercel.app)

# Built With
* HTML5 - Semantic structure and content.
* CSS3 - Custom animations, depth-based styling, and layout.
* Vanilla JavaScript - Scroll-event handling and UI synchronization.

# Project Overview
RAVINE is a visual journey that simulates a descent into the oceanic depths. The project focuses on:
* Synchronized Loading: A custom "Sonar" loader that transitions into the main experience.
* Interactive Depth Levels: Unique visual states triggered by user scroll behavior.
* Performance Optimization: Deployed via Vercel for fast asset delivery and smooth 60fps animations.
